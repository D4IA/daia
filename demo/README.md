## Demo DAIA Mock

Structure created in repo root under `src/` with simple agents, scenarios, and a mock DAIA client.

### Scripts
- `npm run dev`: run directly with ts-node
- `npm run build`: compile to `dist/`
- `npm start`: run compiled output
