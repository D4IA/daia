# Getting started as DAIA developer

## Installation
Install dependencies
```
pnpm i
```

## Start demo
To run all packages
```
pnpm run dev
```

## Build packages
To build everything
```
pnpm run build
```

> Or create Pull Request to main and It'll be handled by CI/CD.

## Test packages
To run tests
```
pnpm run test
```
